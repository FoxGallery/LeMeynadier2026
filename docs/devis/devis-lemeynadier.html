02 · Le projet livré
— 02 — Le

Projet livré

Site vitrine éditorial multilingue, conçu sur-mesure pour la brasserie historique du vieux Cannes. Conçu, développé, testé et documenté.

Design & expérience

  • Identité visuelle « bois noyer · papier crème · laiton » entièrement codifiée.
  • Trois typographies sélectionnées : Cormorant Garamond, Inter Tight, Italianno.
  • Navbar « pill » flottante, menu mobile cinématique plein écran.
  • Animations cinématiques (View Transitions API, GSAP, Lenis).
  • 5 pages publiques : Accueil, Carte, Histoire, Galerie, Nous trouver.
  • Responsive complet — testé sur 5 viewports (360 → 1920 px).

Technique & performance

  • Stack moderne : Nuxt 4, Vue 3.5, TypeScript strict, Tailwind v4.
  • Site 100 % statique (SSG) — compatible n'importe quel hébergement Apache.
  • 192 variantes d'images responsive AVIF/WebP générées automatiquement.
  • Lighthouse CI bloquant : perf ≥ 0.85, accessibilité ≥ 0.95, SEO ≥ 0.95.
  • JSON-LD BarOrPub + Restaurant pour le Knowledge Panel Google.
  • Sitemap multilingue, hreflang, OG images générées à la build.

Sécurité & conformité

  • Headers HTTP durcis (HSTS, CSP, X-Frame, Referrer-Policy, Permissions-Policy) via .htaccess Apache.
  • Aucun cookie publicitaire, aucun tracker tiers, conformité RGPD native sans bannière de consentement.
  • Pipeline CI/CD GitHub Actions : lint, typecheck, tests unitaires (19), E2E Playwright (15+), axe a11y, link-checker, Lighthouse.
« Un site sobre, rapide, beau, et qui se tient bien sur la durée. »
